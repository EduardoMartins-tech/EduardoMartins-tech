## 🚀 Projeto em Destaque

### [CEIControl](https://github.com/EduardoMartins-tech/CEIControl-Alpha-0.3)
> Sistema administrativo web para Centros de Educação Infantil (CEIs) públicos

**🌐 Deploy ao vivo:** [ceicontrol.up.railway.app](https://ceicontrol.up.railway.app)

- ✅ CRUD completo de usuários, produtos, serviços, fornecedores e agenda
- 🔐 Autenticação com 3 perfis de acesso distintos
- 💬 Chat interno em tempo real
- 🌙 Dark Mode nativo
- 🐳 Conteinerizado com Docker e hospedado via Railway
- 🗄️ Banco MySQL com 6 tabelas e relacionamentos
- 🎨 Prototipado no Figma

`PHP` `MySQL` `JavaScript` `HTML5` `CSS3` `Docker` `Railway` `Figma`
